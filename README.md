# Rentify Trust Scorer

An AI-powered renter reliability scoring tool built for India's luxury rental market.

## Problem
India's rental market has a trust problem. Owners renting out luxury items — designer wear, cameras, gadgets — have no way to verify if a renter is reliable before handing over an asset worth ₹10,000–₹2,00,000.

## Solution
Rentify Trust Scorer takes a renter's profile (government ID, rental history, employment proof) and uses AI to generate an instant trust score with risk flags and a rental verdict.
